DMP Help Center

Design System & Component Library — v2.0

1. Design Principles

  • Professional Trust — 外部客户优先，界面传达可靠感。冷色调、干净留白、稳定版式。
  • Scan-friendly — 帮助中心的核心是「快速找到答案」。信息层级清晰，搜索优先。
  • Minimal Distraction — 去除装饰性元素，内容即界面。
  • Consistent — 严格遵循设计令牌，所有间距、色彩、字体统一。

6.4 Sidebar Navigation

基于 Docusaurus 内置侧边栏 + autogenerated 模式。自定义样式：

  • 宽度: 280px (desktop) / 全宽覆盖 (mobile)
  • 背景: #fff，右侧 1px solid #e2e8f0 分隔线
  • 分类标题: 12px / 600 / uppercase / Slate 400 / 12px 16px padding
  • 链接项: 13px / 400 / Slate 600，hover Primary 600 / bg Primary 50
  • 当前激活: Primary 600 / bg Primary 50 / 左侧 2px Primary 600 指示条
  • 嵌套层级: 每级缩进 12px
